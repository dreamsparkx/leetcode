Given two sorted arrays `nums1` and `nums2` of size `m` and `n` respectively, return <b>the median</b> of the two sorted arrays.

The overall run time complexity should be `O(log (m+n))`.

<b>Example 1:</b>

```
Input: nums1 = [1,3], nums2 = [2]
Output: 2.00000
Explanation: merged array = [1,2,3] and median is 2.
```

<b>Example 2:</b>

```
Input: nums1 = [1,2], nums2 = [3,4]
Output: 2.50000
Explanation: merged array = [1,2,3,4] and median is (2 + 3) / 2 = 2.5.
```

```ts title=solution.ts
function findMedianSortedArrays(ar1: number[], ar2: number[]): number {
  let length = ar1.length + ar2.length;
  let index1 = 0,
    index2 = 0;
  let elements = [];
  while (elements.length <= length / 2) {
    if (
      (ar1[index1] !== undefined && ar1[index1] < ar2[index2]) ||
      ar2[index2] === undefined
    ) {
      elements.push(ar1[index1]);
      index1++;
    } else {
      elements.push(ar2[index2]);
      index2++;
    }
  }
  if ((length % 2 === 0 || length === 2) && elements.length >= 2) {
    return (elements[elements.length - 2] + elements[elements.length - 1]) / 2;
  } else {
    return elements[elements.length - 1];
  }
}
```
