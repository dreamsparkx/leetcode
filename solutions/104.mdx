Given the `root` of a binary tree, return its maximum depth.

A binary tree's <b>maximum depth</b> is the number of nodes along the longest path from the root node down to the farthest leaf node.

<b>Example 1:</b>
<br />

<img src={imageURL} width="483" height="342" />

```
Input: root = [3,9,20,null,null,15,7]
Output: 3
```

<b>Example 2:</b>

```
Input: root = [1,null,2]
Output: 2
```

<b>Constraints:</b>

- The number of nodes in the tree is in the range [0, 10<sup>4</sup>].
- `-100 <= Node.val <= 100`

```ts title=TreeNode.ts
class TreeNode {
  val: number;
  left: TreeNode | null;
  right: TreeNode | null;
  constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
    this.val = val === undefined ? 0 : val;
    this.left = left === undefined ? null : left;
    this.right = right === undefined ? null : right;
  }
}
```

```ts title=solution.ts
function maxDepth(root: TreeNode | null): number {
  if (root === undefined || root === null) {
    return 0;
  }
  return Math.max(maxDepth(root.left), maxDepth(root.right)) + 1;
}
```
